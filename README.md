# saving-that-coin - Budget appThis project

A simple personal budget tracker built with Python and Tkinter.
This app allows you to record income, track expenses, set reminders, define goals, and calculate your remining balance.

## Installing / Getting started

make sure you have **Python 3.x** installed:
[Python Download](https://www.python.org/downloads/)

"tkinter" is required (usuall included with Python). if missing: ```bash pip install tk

# Navigate to your project folder
cd saving-that-coin

# Run the main program
python main.py

## Features

-Income Tracking
Enter and save income amounts

-Expense Tracking
Record expenses to keep track of spending

-Reminders
Add custom reminders for financial tasks or due dates

-Goal
save personal financial goals

-Balance Calculator
Automatically calculates

Balance = Total Income - Total Expenses


## Configuration

open the app with "python main.py"
In the income stection, enter your monthly income and click set monthly.
In the expense section, enter a name, amount and date, then click add Expense.
In the remnder section, enter a reminder and click Add Reminder.
In the Goals section, enter goal and click Add goal.
 The app will automatically update the balance when expenses, reminder, or income are added.
#### Argument 1
| Input Field    | Type   | Default | Description                                     |
| -------------- | ------ | ------- | ----------------------------------------------- |
| Monthly Income | Number | 0       | Enter your monthly income in the Income section |
| Expense        | Number | 0       | Enter a name, amount, and date for each expense |
| Reminder       | String | None    | Enter reminders for bills or tasks              |
| Goal           | String | None    | Enter financial goals to track                  |

1.open the app: python main.py

2. In the Income section, enter 3000 and click Set monthly.

3. In the Expense section, enter Rent, 1200, 2025-12-01 and click Add Expense.

4. In the Reminders section, enter Pay credit card, 200, 2025-12-05 and click Add Reminder.

5. In the Goals section, enter Save 500 for vacation and click Add Goal.

The balance is updated automatically after adding income, expenses, reminders, or goals.
